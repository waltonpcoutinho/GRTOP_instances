Instances used in the paper "Trajectory optimisation-based heuristics for routing a fleet of unmanned aerial vehicles"

File format:

x_o y_o h_o (intitial position in Euclidean coordinates)
n (number of waypoints)
x y h \underbar{h} \bar{h} radius (position and geometry of waypoints) for each waypoint
n_l number of landing sites
x y h radius (position and geometry of landing sites)
